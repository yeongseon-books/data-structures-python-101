# data-structures-python-101 예제 코드

Python 내장 자료구조와 stdlib(`collections`, `heapq`, `bisect`)을 활용한 파이썬다운 예제입니다.

## 구성

- 01 what-are-data-structures
- 02 arrays-and-lists
- 03 stacks-and-queues
- 04 hash-tables-and-dict
- 05 linked-lists
- 06 trees-and-binary-trees
- 07 heaps-and-priority-queues
- 08 graph-representations
- 09 sets-and-set-operations
- 10 choosing-data-structures

각 에피소드는 `ko/`와 `en/`에 동일한 실행 가능한 예제 스크립트를 제공합니다.

## 실행

```bash
python ko/01-what-are-data-structures.py
python en/10-choosing-data-structures.py
```

## 테스트

```bash
pip install -r requirements.txt
pytest -q
```
